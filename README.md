# Ejecutar En Desarrollo SHAKTI

1. Clonar repositorio
2. Ejecuta

```
npm install 
```

3. Tener angular cli instalado

```
npm i -g @angular/cli  
```

5. Levantar front ng serve

# Stack Utilizado

* Angular Material
* Boostrap
* Angular CLI
